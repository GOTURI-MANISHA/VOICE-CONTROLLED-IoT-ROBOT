**Voice Controlled IoT Robot** is a smart robotic system that can be controlled using voice commands through the internet.<br>The project uses an ESP32 microcontroller, motor driver, and DC motors to receive commands from a mobile application and perform movements such as forward, backward, left, right, and stop.<br> By integrating IoT and voice recognition technologies, the robot enables wireless and hands-free control, making it useful for robotics learning, smart automation, and remote operation.<br> This project demonstrates how modern communication technologies can be used to create intelligent and user-friendly robotic systems.

Developed this Project for learning smart automation and wireless robot control.
